# coverage_and_incidence_mock_data

Coverage and incidence mock data. Coverage values are computed
considering a sample size of 1234 individuals.

## Usage

``` r
coverage_and_incidence_mock_data
```

## Format

A list with two data frames:

- incidence_data :

  data.frame with weekly incidence data

- coverage_data :

  data.frame with weekly coverage data

## Source

Simulated coverage and incidence data
